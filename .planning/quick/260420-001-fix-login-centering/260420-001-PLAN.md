# Quick Task 260420-001: Fix Login UI Centering

## Root Cause

`app/globals.css` uses Tailwind v3 syntax (`@tailwind base; @tailwind utilities;`) while the project
uses Tailwind v4 (`@tailwindcss/postcss`). This causes CSS class generation to be incomplete —
color, spacing, sizing, and visual utilities are excluded from the compiled CSS bundle.

Compiled CSS was missing: `max-w-md`, `bg-white`, `bg-gray-50`, `rounded-lg`, `shadow-md`, `p-8`,
`space-y-*`, font size classes, etc. The login card had no visual styling.

## Tasks

### Task 1: Fix globals.css Tailwind v4 syntax

**File:** `app/globals.css`  
**Action:** Replace `@tailwind base; @tailwind components; @tailwind utilities;` with `@import "tailwindcss";`  
**Verify:** Dev server rebuild produces CSS with color/spacing/sizing utilities  
**Done:** ✓ Applied
