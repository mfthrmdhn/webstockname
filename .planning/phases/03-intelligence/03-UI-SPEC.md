## Spacing Scale

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline spacing |
| sm | 8px | Compact element spacing, button icon gap |

        <div>

## Interaction Rules

1. **Date range selector is global** — placed in page header, persists across tab switches. State lives in page-level component, passed to each tab as prop.
2. **Tab switch does not trigger a navigation** — use shadcn Tabs (client-side state, no URL change per tab). URL stays `/finance/reports`.
3. **Reconciliation is frontend-only** — no API call on variance calculation. Input onChange triggers immediate recalculation.
4. **Incentive modal blocks background** — Dialog overlay prevents interaction with tab behind it. Clicking outside closes modal only if form is empty; if form has data, clicking outside does nothing (avoid accidental data loss — use `onInteractOutside={(e) => e.preventDefault()}` when form is dirty).
5. **Margin% formula** — `(unitPrice - unitCost) / unitPrice * 100`, weighted across all SaleItems in the sale. Displayed as `XX.X%` (one decimal). If unitCost is null/zero, display `—` instead of calculating.
6. **Superadmin sees Add Incentive button; Finance role does not** — conditional render based on decoded JWT role claim. No empty placeholder div for Finance.
