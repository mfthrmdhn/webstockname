<task type="auto">
  <name>Lock Down Audit Table (Database Immutability)</name>
  <files>prisma/migrations/</files>
  <action>
    Create PostgreSQL trigger to prevent UPDATE/DELETE on audit_log:
    ```sql
    CREATE OR REPLACE FUNCTION raise_immutable_error()
    RETURNS TRIGGER AS $$
    BEGIN
      RAISE EXCEPTION 'audit_log table is immutable';
    END;
    $$ LANGUAGE plpgsql;

    CREATE TRIGGER audit_log_immutable_update
    BEFORE UPDATE ON audit_log
    FOR EACH ROW
    EXECUTE FUNCTION raise_immutable_error();

    CREATE TRIGGER audit_log_immutable_delete
    BEFORE DELETE ON audit_log
    FOR EACH ROW
    EXECUTE FUNCTION raise_immutable_error();
    ```

    Add to Prisma migration or run manually against database.
